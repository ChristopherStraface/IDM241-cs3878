Rules:

  • Term Master Schedule WebTMS button:

    • Hover on:
      • The cursor changes to the finger icon
      • Button changes color to a lighter blue
      • Text becomes underlined


    • Hover off:
      • The cursor reverts to the default pointer
      • Button color reverts to original darker blue
      • The underlined text loses its underline

    • Click:
      • Opens respective page in new tab


  • Search button:

    • Hover on:
      • The cursor changes to the finger icon
      • Button changes color to a lighter blue
      • Text becomes underlined


    • Hover off:
      • The cursor reverts to the default pointer
      • Button color reverts to original darker blue
      • The underlined text loses its underline


    • Click:
      • Takes user to the search results page in the same tab
      • Stores any inputted text from the text-field and puts it in the URL


  • Search text-field:

    • Hover on:
      • The cursor changes to the I-beam pointer
      • Slight shadow added around the text-field


    • Hover off:
      • The cursor reverts to the default pointer
      • Slight shadow around-text field disappears


    • Click on:
      • Type indicator icon starts flickering
      • Black border appears around text-field
      • Sample typing text added


    • Click off:
      • Type indicator icon disappears
      • Black border around text-field disappears
      • Sample typing text removed/Replaced with user-inputted text
