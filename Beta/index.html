Rules:

  • Term Master Schedule WebTMS button:

    • Hover on:
      • The cursor changes to the finger icon
      • Button color fades to a lighter blue in 0.5 seconds
      • An underline fades in below the text in 0.5 seconds


    • Hover off:
      • The cursor reverts to the default pointer
      • Button color fades back to original dark blue in 0.5 seconds
      • The underline below the text fades out in 0.5 seconds

    • Click:
      • Opens respective page in new tab


  • Search button:

    • Hover on:
      • The cursor changes to the finger icon
      • Button color fades to a lighter blue in 0.5 seconds
      • An underline fades in below the text in 0.5 seconds


    • Hover off:
      • The cursor reverts to the default pointer
      • Button color fades back to original dark blue in 0.5 seconds
      • The underline below the text fades out in 0.5 seconds


    • Click:
      • Takes user to the search results page in the same tab
      • Stores any inputted text from the text-field into a database
      • User-inputted text from the text-field gets displayed in the URL


  • Search text-field:

    • Hover on:
      • The cursor changes to the I-beam pointer
      • Slight shadow appears around the text-field in 0.5 seconds


    • Hover off:
      • The cursor reverts to the default pointer
      • Slight shadow around text-field disappears in 0.5 seconds


    • Click on:
      • Type indicator icon starts flickering in 0.5 second intervals
      • Black border appears around text-field in 0.5 seconds
      • Sample typing text gets added to the field in 0.5 seconds


    • Click off:
      • Type indicator icon fades out in 0.5 seconds
      • Black border around text-field fades out in 0.5 seconds
      • Sample typing text fades out in 0.5 seconds OR is replaced by any user-inputted text in the text-field
